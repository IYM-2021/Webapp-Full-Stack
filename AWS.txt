AWS IAM 


Requirements:
node version: > 9.10.0
yarn version: 1.6.0 or >
amplify cli :{ 
	
	      npm install -g @aws-amplify/cli

	      if (first time user)
	      	 amplify init
	      else
		 amplify configure
		 amplify init
	      
}

Node module installation:
npm install 
yarn add aws-amplify aws-amplify-react

Run:
npm start
